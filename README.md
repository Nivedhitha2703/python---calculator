# Simple Calculator in Python

A basic command-line calculator developed using Python. The calculator performs fundamental arithmetic operations based on user input.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division-by-zero handling
- User-friendly interface

## Technologies Used

- Python

## Concepts Demonstrated

- Variables
- User Input
- Conditional Statements (if-elif-else)
- Arithmetic Operators
- Error Handling

## Purpose

This project was created to practice Python fundamentals and understand how conditional logic can be used to build interactive programs.
